# Tenstar Simulation

- **Timeline:** 2024 - Current 
- **Company** Tenstar Simulation
- **Genre:** Vehicle Simulation 
- **Role:** Unreal Engine Programmer

### Overview

## Asphalt Paver Simulation
![](src/images/Paving.gif)
My initial work had a lot to do with actual functionality surrounding the actual paver. Where I had to work with directions from designers, and together with the art department, implement a convincing asphalt laying process.

A lot of time also went into implementing multiplayer functionality using Unreal Engines replication system.

## Machine Visualization
![](src/images/MV.gif)
I was responsible for improving and making it possible for non programmers to setup Machine Inspections, a simulation experience that has even been highlighted in Unreal Engine Simulations Fizzle Reel 2026


## Core Tech
My major contribution to Tenstar has been the creation of the Exercise framework, which allows designers and non programmers alike to create exercises. Using Data layers and State trees for the flow control of the exercise. Along with editor tools to automate the entire file creation proccess.
